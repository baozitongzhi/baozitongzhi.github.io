# Airplanes

Aviation • Flow Visualization • Engines

I am documenting my learning journey in aviation visualization, starting with Blender-based airflow and vapor cloud experiments around an Airbus A340-600 model.

## Current Stage

### Stage 1 — Blender Flow Visualization

Completed:
- Imported and prepared an A340-600 model
- Learned basic Blender navigation and object editing
- Created a gas domain for smoke simulation
- Set up inflow objects
- Added wind force fields
- Tested smoke density, transparency, vorticity, gravity, and cache settings
- Learned the difference between visual smoke simulation and real CFD
- Began experimenting with vapor-cloud-style airflow visualization

## Main Project

### A340-600 Vapor Cloud Visualization

Goal:

To visually recreate the condensation cloud and wake-like vapor seen behind the wing of an Airbus A340-600 during humid flight conditions.

Current status:

Early Blender visualization stage.
